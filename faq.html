Before you can download Dosware you must check if you have these requirements

You will need these programs/software to run dosware corectlly

  • A 64 bit Installation of Windows 10

  • VC Redist x64

  • VC Redist x86

  • .NET Framework 4.7.2+

  • Web version of roblox

  1. To check that you have Windows 10 and it's 64bit

    • Have more than 3 braincells

    • Type 64bit into the search bar in Windows 10. It should pop up with a suggestion saying "Is my PC 32bit or 64bit".

      • Click on it and see your Operating System architecture.

  2. To check that you have the VC Redists

    1. Go to the "Add or remove programs" section in the Windows 10 settings.

    2. Then search for Visual C++ Redistributable.

    3. If two programs named along the lines of Visual 2015-2019 x64 and x86 show up that means you have it.

  3. To check that the .NET package is installed

    1. Open command prompt and type "dotnet" and hit enter

    2. If it says there is no such command it means its not installed.

    3. Now use your braincells and go download .net

  4. To check that the webclient version of Roblox is installed/install it.

    1. Uninstall the Microsoft Store version of Roblox if you have it

    2. Install the web version from the roblox website. Link is in the Requirements above

1.2 Antivirus blocking downloading/running Dosware

Why this happens

  • Antiviruses usually mistake Roblox exploits as viruses due to their nature. Most if not all modern Roblox exploits work by injecting some sort of dll file into Roblox and running Lua code (Scripts) in the environment it creates.

  • But heres the catch: Dll injection isnt just used by game cheats. They are also used by malware. Due to this antivirus try to quarantine/delete all dll injectors they detect without actually knowing what if they are safe or not. You will usually see antivirus calling most exploits including dosware a Cryptinject.msr trojan because of this.

  • "But then how can you prove Dosware isn't malware?" you might ask. User trust and safety is priority at Dosware, due to this the only hidden/protected part of the Dosware client is the dll that is injected into Roblox(This being easyexploits.dll). Other than the dll Dosware client is completely Safe. Dosware's code is completely readable, using a .NET decompiler like dnSpy or DotPeek but dont skid it. This means that you yes you can look at the code that runs the Dosware client.

    Please keep in mind that this isnt the exact source code as the Asyncronous Functions wont be fully decompiled in dnSpy. However it should still give you a good idea of if Dosware is safe or not.

    SKIP THIS STEP IF YOU DONT HAVE A 3RD PARTY ANTIVIRUS

    To fix this issue you must uninstall any 3rd party antivirus from your system. Such as McAfee, Norton, Eset, Kaspersky etc. Sorry but there isnt another way of solving this issue. From my and many users experiences excluding/disabling 3rd party antivirus is terrible and doesnt yield consistent/reliable results.

    Once that you have uninstalled your 3rd party antivirus and rebooted your pc (Or you didnt have any to begin with). Its time to exclude Dosware from Windows Defender which is the default antivirus that comes with Windows 10. To do this we recommend watching a tutorial from YouTube. But in short all you need to do is:

    • Go to the virus and threat protection settings in Windows 10.

    • Click on manage settings.

    • Scroll to the bottom where it says "Exclusions".

    • Click on add an exclusion then choose the option: "Folder" from the dropdown menu.

    • After that click on "This PC" and whitelist your whole C drive.
    • By doing this you shouldnt have any troubles using Dosware at all.
